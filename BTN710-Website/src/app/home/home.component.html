Process:

Steps taken to encrypt the password:

  • We first hashed the password using the bcrypt library via Node.js
  • We used console.log to access the hash password
  • After that, we then took the hashed password from the console and placed it into an Angular Service
  • We use bcrypt's compareSync functions to compare the user inputted code against the hashed password
  • If the password was the same, we then authenticated the user - otherwise we told them that their entered password was incorrect
